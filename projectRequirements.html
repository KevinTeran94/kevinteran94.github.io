Project Requirements

The above is a low level business Context Diagram.

Requirement Scope Area Description
Tea Subscription Box App A mobile application works as a tea subscription box and collect information to provide detailed statistics to management.
External Entity Description
Admin/Business Account Administration team who manages, maintains and updates the application
Guest/Unregistered User Guest/Unregistered User can view the business content and register customer account
Customer/Registered User Customer/Registered User can view content, edit profile, place order, and give feedback about the product and service
Information Flows Description
Maintain and update Administration team
Generate analysis/statistics Analytics/Statistics are generated by the application based on database
Register account Guest user can be able to sign up for new customer account
View contents User can access to the application content
Edit profile Customer/Registered User can manage and update their account profile
Place order Customer/Registered User can place order by subscribe the plan
Give feedback Customer/Registered User can leave feedback based on their purchase experience
Generate order detail Customer/Registered User can view their receipt generated by the application
